** Firefox 相关扩展

** 开始使用我的配置
只有单个配置文件，这一步比较简单。当然你也可以在线参照，为了照顾自己的感受，请先备份已存在的配置文件。
: wget https://raw.githubusercontent.com/oxh/oconfig/master/keysnail/keysnail.js.bak
: mv keysnail.js.bak ~/.keysnail.js
